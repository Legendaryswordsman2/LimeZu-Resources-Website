Modern User Interface

(Status: Actively added to every once in a while)

Modern User Interface is the third entry in the Modern Series, it was one of the most anticipated additions to the series. This pack contains all sorts of UI such as buttons, toggles, fill bars, and anything else you may need for you UI.

Note: most of the Modern User Interface pack is contained inside of one or two png files so this page won't be a guide for where everything is, instead it will act more of an overview of everything included.

Icons

Modern UI contains a variety of UI icons including the following (Note this is not an exhaustive list):

-Common Social Media icons (YouTube, Twitter, Facebook, Discord, etc)
-Two variations of arrows facing in all four direction
-Various common UI icons such as a trash can icon, check mark and X icons, settings icon, mute/unmute audio icon, play/stop icons, etc.
-Money icons.
-Emote faces.
-Stars.
